# Core Analysis Instructions

## Data Preprocessing
- Validate data quality and completeness before analysis
- Check for and handle missing values appropriately
- Identify and address outliers based on statistical methods
- Normalize or standardize data when necessary
- Convert data types to appropriate formats

## Statistical Analysis
- Calculate basic descriptive statistics (mean, median, mode, standard deviation)
- Identify significant patterns and trends in the data
- Perform correlation analysis between relevant variables
- Test for statistical significance when making comparisons
- Apply appropriate statistical tests based on data distribution

## Pattern Recognition
- Look for recurring patterns or cycles in time-series data
- Identify anomalies that deviate from expected patterns
- Group similar data points using clustering techniques
- Detect relationships between different variables
- Flag unusual or unexpected correlations

## Reporting Guidelines
- Present findings in clear, concise language
- Support conclusions with specific data points
- Provide confidence levels for predictions
- Include visualizations for key insights
- Highlight actionable recommendations

## Error Handling
- Document assumptions made during analysis
- Note limitations of the analysis methods used
- Flag potential data quality issues
- Provide confidence intervals where applicable
- Identify areas requiring human review

## Priority Framework
1. Critical insights that require immediate attention
2. Long-term trends affecting business objectives
3. Potential opportunities for optimization
4. Risk factors and warning signs
5. Areas needing further investigation

## Output Format
- Executive summary limited to key findings
- Detailed analysis section with supporting data
- Visual representations of critical metrics
- Recommendations section with clear action items
- Technical appendix with methodology details
